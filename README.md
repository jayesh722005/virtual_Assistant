## рҹҡҖ Installation

### Clone Repository

```bash
git clone https://github.com/jayesh722005/virtual_Assistant.git

cd virtual_Assistant
```

---

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on

```
http://localhost:5173
```

---

### Install Backend

```bash
cd backend
npm install
npm start
```

Runs on

```
http://localhost:8000
```
